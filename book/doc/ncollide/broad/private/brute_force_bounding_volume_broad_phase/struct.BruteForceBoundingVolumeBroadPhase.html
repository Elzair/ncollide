Broad phase with quadratic complixity but sped up using looze bounding volumes. Interference detection is executed only for objects which have their bounding volumes updated. The number of such objects is assumed to be small enough to not exhibit the quadratic nature of the detection algorithm.

Dont use this broad phase. It exists mainly as a transition broad phase between the Brute Force one and the DBVH/SAP based broad phases.
