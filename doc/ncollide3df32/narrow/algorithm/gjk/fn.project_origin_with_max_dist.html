pub fn project_origin_with_max_dist<S: Simplex<_V>, G: Implicit<_V, _M>, _V: FloatVec<Scalar>, _M>(m: &_M, geom: &G, max_dist: &Scalar, simplex: &mut S) -> GJKResult<_V, _V>

Projects the origin on a geometry using the Separating Axis GJK algorithm. The algorithm will stop as soon as the polytope can be proven to be at least max_dist away from the origin.

Arguments:

  • geom - the geometry to project the origin on
  • simplex - the simplex to be used by the GJK algorithm. It must be already initialized with at least one point on the geometry boundary.
