Function ncollide::narrow::algorithm::gjk::project_origin_with_max_dist

pub fn project_origin_with_max_dist<S: Simplex<N, V>, G: Implicit<N, V, M>, N: Ord + Num + Float + Cast<f32>, V: AlgebraicVec<N> + Clone, M>(m: &M, geom: &G, max_dist: &N, simplex: &mut S) -> GJKResult<V, V>

Projects the origin on a geometry unsing the Separating Axis GJK algorithm. The algorithm will stop as soon as the polytope can be proven to be at least max_dist away from the origin.

Arguments:

* geom - the geometry to project the origin on
* simplex - the simplex to be used by the GJK algorithm. It must be already initialized
with at least one point on the geometry boundary.
