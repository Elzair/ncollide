Required Methods

fn contains_point(&self, point: &Vect) -> bool

Tests whether or not self contains a given point.

fn project_point(&self, point: &Vect, limit: &Scalar) -> Option<PointProjection>

Computes the closest projection of a point on self. The closest projection is the global minimizer of the distance function between the point and the object. Returns None if there is no projection or if the projection is further than limit.

fn closest_points(&self, point: &Vect, limit: &Scalar) -> Vec<PointProjection>

The closest projection is the set of local minimizers of the distance function between the point and the object. Returns None if there is no projection or if the projection is further than limit.
